## 3. 제목 작성 원칙 (v1에서 수정)

**v1의 문제**: 제목을 경험형으로만 바꾸면 검색 유입이 사라짐.

**v2 원칙**: SEO 검색어(앞) + 경험 후킹(뒤) 조합

| ❌ v1 방식 | ✅ v2 방식 |
|-----------|-----------|
| 니혼바시에서 외국인이 은행 계좌를 만들기까지 | 일본 은행 계좌 개설 방법 — 니혼바시에서 실제로 겪은 경험 |
| 니혼바시로 이사 와서 처음 당황했던 것 | 일본 쓰레기 분리수거 방법 — 니혼바시로 이사 와서 가장 당황했던 것 |
| 니혼바시 거주자가 아는 카페 | 니혼바시 카페 추천 — 2년째 살면서 찾아낸 단골 5곳 |

**공식**: `[검색 키워드] — [경험/현실 후킹]`

## 5. 사진 원칙 (v2 신규 추가)

**사진은 선택이 아니라 필수입니다.**
AI 글과 실제 운영자 글을 구분하는 가장 빠른 신호.
AdSense 심사관도 사진으로 "실제 운영 사이트"를 판단.

### 글별 사진 요구사항

- 각 글 최소 1~2장 실제 사진 포함
- 가능한 경우 목사님이 직접 촬영한 사진 사용

### 니혼바시 사진 소재 목록

- 니혼바시 거리/강변
- 코레도 무로마치
- 단골 카페 내부
- 슈퍼마켓 내부
- 실제 거주 콘도 주변
- 일상 장면 (쓰레기 분리수거 스티커, 마트 영수증 등)

> GPT 작업 시: 사진이 없는 글은 `[사진 필요: 니혼바시 ○○ 촬영]` 표시를 본문에 남겨주세요.

일본에서 실제로 살아가며 겪은 경험을 기록하고, 일본 생활을 준비하는 분들에게 현실적인 정보를 전하기 위해 만들었습니다.

## 7. TASK B — Tier 1 신규 5편 (경험 에세이)

기존 15편보다 중요. About과 함께 AdSense 심사의 핵심.

### 편 1: 왜 도쿄 니혼바시에 살게 되었나
소재: 치앙마이→일본행 결정 / 니혼바시 선택 이유 / 도로원표 상징성
톤: 따뜻하고 진솔한 에세이

### 편 2: 니혼바시에 집을 사기까지 — 외국인 부동산 매입 실전기
소재: 외국인 모기지 현실 / 購入申込書 제도 / 타이밍 긴박함 / 실제 비용
톤: 실용 정보 + 경험담
⭐ TokyoKorean 대표 글 후보

### 편 3: 일본인 아내와 살며 놀랐던 문화 차이
소재: 한일 가족문화 / 집안일·식문화 / 서로에게 배운 것
톤: 유머 있게, 어느 한쪽 비하 없이

### 편 4: 개호 현장에서 본 일본의 고령사회
소재: 킹스가든 근무 경험 / 일본 노인 생활방식 / 한국 복지와 차이
톤: 진지하고 따뜻하게. 선교색 없이.

### 편 5: 일본 생활 7년 차 한국인이 솔직하게 말하는 것들
소재: 좋았던 것/힘들었던 것 / 한국 돌아가고 싶었던 순간 / 그래도 있는 이유
톤: 솔직하고 현실적으로. TokyoKorean의 "심장"
