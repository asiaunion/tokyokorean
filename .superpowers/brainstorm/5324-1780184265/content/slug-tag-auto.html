게시글 설정 패널 — 자동화 기능

한글 제목에서 영문 주소를 자동 생성하고, 본문 내용에서 태그를 자동으로 추천해주는 기능이에요.

현재 방식 — 수동 입력
⚙️ 게시글 설정
웹 주소 (영문)
영문 소문자, 숫자, 하이픈(-)만 사용
태그
태그 없음
⚠️ 영문 주소를 직접 타이핑해야 함.
태그도 하나씩 수동으로 입력 필요.
✅ 자동 생성 + AI 태그 추천
⚙️ 게시글 설정
웹 주소 (영문)
← "도쿄 신주쿠 여행 가이드"에서 자동 변환됨
태그
도쿄 신주쿠 여행
✨ AI 추천 태그 (클릭하면 추가):
맛집 관광 일본 가이드
✅ 제목 입력 → 영문 주소 자동 변환 (직접 수정도 가능)
✅ 본문 저장 시 AI가 태그 추천 → 클릭해서 추가

🔧 동작 방식 상세

1

슬러그 자동 변환만

한글 → 로마자 변환(romanization) 후 kebab-case로. 예: "신주쿠 여행" → "shinjuku-travel". 추가 API 불필요, 순수 JS로 처리.

2

슬러그 자동 변환 + AI 태그 추천 둘 다 ✅

슬러그는 JS 로마자 변환. 태그는 저장 시 LLM(Gemini/OpenAI)이 본문 분석 후 5개 이내 추천. 기존 .env에 API 키가 있어서 바로 연동 가능.
