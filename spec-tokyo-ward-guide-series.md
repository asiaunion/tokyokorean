# Spec: 도쿄 지역 가이드 시리즈 — GSF-Blog

## Overview

이주·투자 검토 중인 한국인·외국인을 핵심 타깃으로, 도쿄 23구 및 多摩 주요 시를
이미지/브랜드 기준으로 그루핑하여 총 **12편** 시리즈로 발행한다.  
각 편은 부동산 데이터(매매가·임대가·소득)를 최우선으로, 동네 포지셔닝과 관광 정보를 보완한다.

---

## Goals

- 도쿄 이주·투자 검토 독자에게 지역별 입지 포지셔닝 + 실거래 데이터를 원스톱으로 제공
- 시리즈 구조로 SEO 클러스터 형성 (개요편 = 허브, 각 그룹편 = 클러스터)
- 관광 정보를 보조 레이어로 추가해 일반 독자 유입도 확보
- 향후 "노선/지리 기반", "가격대 기반" 시리즈의 파운데이션 역할

---

## Scope

**In:**
- 도쿄 특별구 23구 전체
- 多摩 지역 주요 시 (무사시노·미타카·조후·하치오지·마치다·다마·후추·고쿠분지·니시도쿄 등)

**Out:**
- 도서부(大島·八丈島 등)
- 가나가와·사이타마·치바 등 도쿄 인접 현
- 일본 전국 부동산 비교 (도쿄 내부에 집중)

## 시리즈 구성 — 총 12편

### Episode 0 (개요편) — 시리즈 프롤로그
**제목 방향**: "도쿄 어디에 살 것인가 — 23구 + 多摩 완전 가이드 시작"  
**역할**: 허브 포스트. 전체 지역 지도 개요, 브랜드 그루핑 논리 설명, 나머지 11편 예고 링크.  
**발행 방식**: Ep0 **선발행** 후 Ep1~Ep11 순차 발행. 개요편에 전편 링크 업데이트.  
**핵심 콘텐츠**:
- 도쿄 23구 한눈에 보기 (가격대·성격 매트릭스)
- 핵심 6구(千代田·中央·港·新宿·渋谷·文京) 개념 설명
- "이 시리즈를 읽어야 하는 독자" 세 유형 (이주자 / 투자자 / 여행자)
- 브랜드 기준 그루핑 설명 + 全 에피소드 예고 목록

## 각 편 콘텐츠 템플릿

> 분량 제약 시 **우선순위 상위 항목**부터 수록.

> 목표 글자수: **KO 기준 2,000자+** (Ep1 핵심 3구는 구별 심층 분석으로 제한 없이 길어도 무방)

### Priority 1 — 투자·이주 핵심 (必수)
1. **동네 브랜드 포지셔닝** — "이 지역 = OO 이미지" 한 문단 요약
2. **맨션 평균 매매가** — ㎡ 단가 (坪 단가 병기), 최근 3~5년 추이
3. **임대 평균 시세** — 1R·1K·1LDK·2LDK 등 타입별 월세 범위
4. **평균 세대 소득** — 구 평균 연수입 (도쿄 평균 대비 지수)

### Priority 2 — 생활 맥락 (중요)
5. **인구수** — 총인구, 인구 밀도, 최근 증감 트렌드
6. **외국인 비율** — 외국인 주민 비율, 한국인 커뮤니티 유무
7. **추천 타깃 독자** — 이주자 / 투자자 / 여행자 각 1줄 결론

### Priority 3 — 관광·라이프스타일 (보완)
8. **대표 관광지·명소** — TOP 3~5개, 이주자 관점 메모 병행
9. **동네 분위기 스케치** — 1~2 paragraph, 걷는 느낌, 대표 상권

### Priority 4 — 투자 유의사항 (마지막)
10. **외국인 매입 주의사항** — 행정 언어 지원, 외국인 토지 취득 제한 여부, 관리비·수선적립금 이슈

## Technical Notes

- **언어**: KO 원문 → deploy-blog 스킬로 EN/JA 자동 번역
- **배포 워크플로 (필수)**: KO 초고 작성 → 사용자 검토 승인 → EN/JA 번역 → Git Push. 별도 지시가 없는 한 **배포는 검토 승인 후에만 실행**.
- **데이터 출처**: 東京都統計, LIFULL HOME'S, SUUMO, 国税庁 민간급여통계, 各区 세대 소득 데이터
- **이미지**: `generate_image` 툴로 지역 분위기 일러스트 생성, WebP 변환
- **내부 링크**: 개요편(Ep0)이 나머지 전편 허브. 각 편에서 Ep0로 역링크.
- **기존 포스트 충돌**: `tokyo-office-vacancy-five-wards-2026.md`(오피스 공실) 및 `tokyo-mansion-tsubo-chiyoda-chuo-minato.md`(坪 단가)와 내용 겹침 주의 → 해당 포스트로 내부 링크 처리, 중복 최소화

## Tradeoffs and Decisions

| 결정 | 선택 | 이유 |
|------|------|------|
| 그루핑 기준 | 이미지/브랜드 | 독자가 "나는 어떤 동네 스타일?" 관점으로 탐색하기 쉬움. 가격대/노선 기준은 추후 별도 시리즈 |
| 포스트 구조 | 개요편 + 그룹별 심화편 | SEO 허브-클러스터 구조, 개요편이 전체 트래픽 앵커 역할 |
| 多摩 처리 | 동일 브랜드 기준, 압축 | 다마는 23구 대비 인지도 낮음. 시들을 묶어 편당 3~6개 시 처리 |
| 콘텐츠 우선순위 | 투자/이주 데이터 우선 | 타깃 독자의 핵심 pain point. 관광 정보는 보조 레이어 |

---

## Failure Modes

- **데이터 노후화**: 매매가·임대가는 연 1회 이상 업데이트가 필요. 각 편에 "데이터 기준 시점" 명기 필수
- **구 단위 일반화 오류**: 世田谷 등 넓은 구는 서브 지역(오카야마/후타코타마가와 등) 편차 큼 → 주요 역세권 기준으로 명확히 한정
- **기존 포스트 중복**: 기발행 坪단가·오피스 공실 포스트와 내용 겹침 → 링크 처리, 불필요한 중복 회피
- **번역 품질**: 부동산 일본어 전문 용어(借地権, 管理費, 修繕積立金 등) 번역 시 검수 필요
